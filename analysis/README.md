# Analysis Prompts

Prompts for data analysis, research, problem-solving, and critical thinking.

## When to Use These Prompts

Use analysis prompts when you need help with:
- Analyzing data and identifying patterns
- Solving complex problems systematically
- Making data-driven decisions
- Conducting research and synthesis
- Evaluating options and trade-offs
- Understanding trends and insights

## Available Prompts

- **[Data Analysis Assistant](data-analysis.md)** - Analyze datasets and generate actionable insights
- **[Problem Solver](problem-solver.md)** - Structured approach to complex problem-solving

## Tips for Analysis Prompts

- Provide sufficient context and background
- Include relevant data or information
- Be specific about what you want to learn
- Mention any constraints or requirements
- Define your goals and success criteria
- Specify the format you prefer for results

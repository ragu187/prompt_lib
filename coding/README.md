# Coding Prompts

Prompts for software development, debugging, code review, and technical documentation.

## When to Use These Prompts

Use coding prompts when you need help with:
- Writing or reviewing code
- Debugging errors and issues
- Generating documentation
- Understanding codebases
- Refactoring and optimization
- Testing and quality assurance
- Learning new languages or frameworks

## Available Prompts

- **[Code Review Assistant](code-review.md)** - Comprehensive code reviews focusing on quality, security, and best practices
- **[Debug Helper](debug-helper.md)** - Systematic debugging for identifying and fixing issues
- **[Function Documentation Generator](function-documentation.md)** - Generate comprehensive function documentation

## Tips for Coding Prompts

- Include relevant code context, not just isolated snippets
- Specify the programming language and framework
- Mention any coding standards or style guides you follow
- Provide error messages in full (including stack traces)
- Describe what you've already tried
- Include version information for languages and tools

# Reasoning and Problem-Solving Prompts for Anthropic Claude

## Complex Problem Solving

```
You are an expert problem solver with strong analytical and creative thinking abilities.

Problem: [Describe the problem]

Context: [Provide relevant background]

Approach this systematically:
1. Break down the problem into components
2. Analyze constraints and requirements
3. Generate multiple solution approaches
4. Evaluate pros and cons of each approach
5. Recommend the best solution with justification
6. Identify potential risks and mitigation strategies
```

## Logical Reasoning

```
You are a logic expert skilled in formal reasoning.

Statement/Argument: [Present the logical problem or argument]

Tasks:
1. Identify the logical structure
2. Evaluate validity of reasoning
3. Identify any logical fallacies
4. Assess the strength of conclusions
5. Provide counterarguments if applicable
```

## Ethical Dilemma Analysis

```
You are an ethics expert trained in philosophical reasoning.

Scenario: [Describe ethical dilemma]

Analyze from multiple ethical frameworks:
1. Consequentialism (outcomes)
2. Deontology (duties/rules)
3. Virtue ethics (character)
4. Care ethics (relationships)

For each framework:
- Apply the principles
- Reach a conclusion
- Compare perspectives
- Recommend a balanced approach
```

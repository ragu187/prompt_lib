# Creative Writing Prompts for OpenAI Models

## Story Generation

```
You are a creative storyteller with expertise in [genre].

Write a compelling story with the following elements:
- Setting: [describe setting]
- Characters: [describe main characters]
- Conflict: [describe central conflict]
- Tone: [e.g., suspenseful, humorous, dramatic]

Create an engaging narrative with vivid descriptions and natural dialogue.
```

## Content Brainstorming

```
You are a creative content strategist.

Topic: [Your topic]
Target Audience: [Describe audience]
Format: [blog post, video script, social media, etc.]

Generate 10 unique and engaging content ideas that would resonate with the target audience.
```

## Character Development

```
Create a detailed character profile for a fictional character:

Basic Info:
- Name: [Character name]
- Role: [protagonist, antagonist, supporting]
- Story context: [brief story summary]

Develop:
1. Physical appearance
2. Personality traits
3. Background/history
4. Motivations and goals
5. Flaws and conflicts
6. Character arc
```

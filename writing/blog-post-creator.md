# Blog Post Creator

## Category
Writing

## Description
Create engaging blog posts on any topic with proper structure, SEO considerations, and audience engagement.

## Prompt

```
Create a blog post on the following topic:

**Topic:** [TOPIC]

**Requirements:**
- Target Word Count: [WORD_COUNT]
- Tone: [TONE]
- Target Audience: [AUDIENCE]
- Key Points to Cover: [KEY_POINTS]

**Structure:**
1. Catchy title
2. Engaging introduction with hook
3. Main content with subheadings
4. Actionable takeaways
5. Compelling conclusion with call-to-action

**SEO Keywords:** [KEYWORDS]

Please make it informative, engaging, and valuable to readers.
```

## Variables

- `[TOPIC]`: The subject of the blog post
- `[WORD_COUNT]`: Target length (e.g., 500, 1000, 2000 words)
- `[TONE]`: Writing tone (informative, conversational, professional, etc.)
- `[AUDIENCE]`: Who will read this post
- `[KEY_POINTS]`: Main points or arguments to include
- `[KEYWORDS]`: SEO keywords to incorporate naturally

## Example Usage

### Input
```
Create a blog post on the following topic:

**Topic:** Benefits of Remote Work for Developers

**Requirements:**
- Target Word Count: 1000 words
- Tone: Conversational and informative
- Target Audience: Software developers and tech professionals
- Key Points to Cover: Flexibility, productivity, work-life balance, cost savings, global opportunities

**Structure:**
1. Catchy title
2. Engaging introduction with hook
3. Main content with subheadings
4. Actionable takeaways
5. Compelling conclusion with call-to-action

**SEO Keywords:** remote work, software development, productivity, work from home

Please make it informative, engaging, and valuable to readers.
```

### Expected Output
A complete blog post with title, well-structured content, subheadings, and engaging writing that covers all key points.

## Tips

- Provide specific details about your target audience
- List the main points you want covered
- Specify the desired tone and style
- Include any specific examples or data to incorporate
- Mention any links or resources to reference
- Can be adapted for different blog styles (listicles, how-tos, opinion pieces)

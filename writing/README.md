# Writing Prompts

Prompts for content creation, editing, technical writing, and storytelling.

## When to Use These Prompts

Use writing prompts when you need help with:
- Creating various types of content (blogs, articles, documentation)
- Editing and improving existing text
- Crafting emails and communications
- Writing marketing copy
- Technical documentation
- Creative writing

## Available Prompts

- **[Content Improvement Editor](content-improvement.md)** - Enhance text for clarity, engagement, and correctness
- **[Blog Post Creator](blog-post-creator.md)** - Create engaging blog posts with SEO optimization

## Tips for Writing Prompts

- Specify your target audience
- Define the desired tone (professional, casual, formal, friendly)
- Include the purpose of the writing
- Mention any word count requirements
- Provide context about the subject matter
- Specify any style guides or formats to follow

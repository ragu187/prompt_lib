# Content Improvement Editor

## Category
Writing

## Description
Enhance and improve written content for clarity, engagement, grammar, and style while maintaining the original message.

## Prompt

```
Please improve the following text:

[TEXT_TO_IMPROVE]

Focus on:
1. **Grammar & Spelling**: Fix any errors
2. **Clarity**: Make the message clearer and easier to understand
3. **Conciseness**: Remove unnecessary words while keeping meaning
4. **Engagement**: Make it more engaging and interesting
5. **Tone**: Adjust tone to be [TONE]

Target Audience: [AUDIENCE]
Purpose: [PURPOSE]

Please provide:
- The improved version
- A brief explanation of key changes made
```

## Variables

- `[TEXT_TO_IMPROVE]`: The text that needs improvement
- `[TONE]`: Desired tone (professional, casual, formal, friendly, etc.)
- `[AUDIENCE]`: Who will read this content
- `[PURPOSE]`: What the content aims to achieve

## Example Usage

### Input
```
Please improve the following text:

Hey, so like, we're basically trying to make this thing work better and stuff. The users were kind of complaining that it's too slow and confusing. We should probably fix that soon maybe.

Focus on:
1. **Grammar & Spelling**: Fix any errors
2. **Clarity**: Make the message clearer and easier to understand
3. **Conciseness**: Remove unnecessary words while keeping meaning
4. **Engagement**: Make it more engaging and interesting
5. **Tone**: Adjust tone to be professional

Target Audience: Project stakeholders
Purpose: Status update

Please provide:
- The improved version
- A brief explanation of key changes made
```

### Expected Output
The AI will provide a professional, clear version and explain how it improved the tone and clarity.

## Tips

- Be specific about the desired tone and style
- Provide context about the audience and purpose
- Can be used for emails, reports, articles, social media posts
- Ask for multiple versions if you want options
- Specify any constraints (word count, key points to include)

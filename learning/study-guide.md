# Study Guide Generator

## Category
Learning

## Description
Create comprehensive study guides for any topic with key concepts, practice questions, and learning strategies.

## Prompt

```
Create a study guide for:

**Subject/Topic:**
[SUBJECT]

**Scope:**
[SPECIFIC_TOPICS_OR_CHAPTERS]

**Learning Goals:**
[WHAT_YOU_NEED_TO_MASTER]

**Current Knowledge Level:**
[YOUR_LEVEL]

**Time Available:**
[STUDY_TIMELINE]

**Learning Style Preference:**
[VISUAL/AUDITORY/READING/KINESTHETIC]

Please create a study guide with:
1. **Key Concepts Overview**: Main topics and their relationships
2. **Concept Breakdown**: Detailed explanation of each concept
3. **Important Terms & Definitions**: Glossary of key terms
4. **Practice Questions**: Mix of multiple choice, short answer, and application questions
5. **Study Strategies**: Effective ways to learn this material
6. **Common Pitfalls**: What students typically struggle with
7. **Real-World Applications**: How this knowledge is used
8. **Memory Aids**: Mnemonics, analogies, or tricks to remember
9. **Study Schedule**: Suggested timeline for covering material
10. **Self-Assessment**: How to test your understanding
```

## Variables

- `[SUBJECT]`: The subject or topic to study
- `[SPECIFIC_TOPICS_OR_CHAPTERS]`: What specifically to focus on
- `[WHAT_YOU_NEED_TO_MASTER]`: Learning objectives or exam requirements
- `[YOUR_LEVEL]`: Current knowledge level (beginner, intermediate, advanced)
- `[STUDY_TIMELINE]`: How long until test/deadline
- `[LEARNING_STYLE]`: How you learn best

## Example Usage

### Input
```
Create a study guide for:

**Subject/Topic:**
JavaScript Promises and Async/Await

**Scope:**
Asynchronous JavaScript, Promises, async/await syntax, error handling, Promise methods

**Learning Goals:**
Master async programming for upcoming technical interview

**Current Knowledge Level:**
Comfortable with JavaScript basics, some experience with callbacks

**Time Available:**
2 weeks

**Learning Style Preference:**
Learning by doing - prefer examples and exercises

Please create a study guide with:
1. **Key Concepts Overview**: Main topics and their relationships
2. **Concept Breakdown**: Detailed explanation of each concept
3. **Important Terms & Definitions**: Glossary of key terms
4. **Practice Questions**: Mix of multiple choice, short answer, and application questions
5. **Study Strategies**: Effective ways to learn this material
6. **Common Pitfalls**: What students typically struggle with
7. **Real-World Applications**: How this knowledge is used
8. **Memory Aids**: Mnemonics, analogies, or tricks to remember
9. **Study Schedule**: Suggested timeline for covering material
10. **Self-Assessment**: How to test your understanding
```

### Expected Output
A comprehensive guide with concept maps, code examples, practice problems, common interview questions, and a day-by-day study plan.

## Tips

- Be specific about what you need to learn and why
- Mention any specific exam or assessment format
- Include your timeline to get appropriate depth
- Specify areas you find most challenging
- Request specific types of practice questions
- Can be adapted for academic, professional, or skill-based learning

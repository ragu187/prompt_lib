# Learning Prompts

Prompts for studying, understanding concepts, educational purposes, and skill development.

## When to Use These Prompts

Use learning prompts when you need help with:
- Understanding complex concepts
- Studying for exams or certifications
- Learning new skills or technologies
- Breaking down difficult topics
- Creating study materials
- Practicing with examples
- Reinforcing understanding

## Available Prompts

- **[Concept Explainer](concept-explainer.md)** - Get clear explanations of complex concepts with examples
- **[Study Guide Generator](study-guide.md)** - Create comprehensive study guides with practice questions

## Tips for Learning Prompts

- Be honest about your current knowledge level
- Specify what you find confusing
- Ask for examples from familiar domains
- Request different levels of detail
- Mention your learning style preferences
- Include your timeline and goals
- Ask for practice problems or exercises
- Request connections to related concepts

# Multimodal Prompts for Google Models (Gemini)

## Image Analysis

```
You are an expert in visual analysis and interpretation.

[Image would be provided here]

Analyze the image and provide:
1. Detailed description of contents
2. Context and setting
3. Notable features or patterns
4. Potential significance or meaning
5. Technical aspects (composition, lighting, etc.)
```

## Document Understanding

```
You are a document analysis expert.

[Document image/PDF would be provided here]

Extract and analyze:
1. Document type and purpose
2. Key information and data points
3. Structure and organization
4. Important sections or highlights
5. Summary of main content
```

## Visual Question Answering

```
You are an AI assistant skilled at answering questions about visual content.

[Image would be provided here]

Question: [Specific question about the image]

Provide:
- Direct answer to the question
- Supporting visual evidence
- Relevant context
- Confidence level in the answer
```

# Classification Prompts for Cohere Models

## Text Classification

```
You are a text classification expert.

Classification Task: [Describe what to classify]
Categories: [List possible categories]

Text to classify:
[Input text]

Provide:
1. Primary category
2. Confidence score
3. Reasoning for classification
4. Alternative categories if applicable
```

## Sentiment Analysis

```
You are a sentiment analysis specialist.

Text to analyze:
[Input text]

Analyze and provide:
1. Overall sentiment (positive/negative/neutral)
2. Sentiment intensity (scale 1-10)
3. Key phrases indicating sentiment
4. Emotional tone
5. Nuances or mixed sentiments
```

## Intent Detection

```
You are an intent detection expert for conversational AI.

User Input: [User's message]

Context: [Conversation context if available]

Identify:
1. Primary intent
2. Secondary intents if present
3. Required entities/parameters
4. Confidence level
5. Suggested response strategy
```

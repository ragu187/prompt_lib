# Productivity Prompts

Prompts for task management, planning, organization, and workflow optimization.

## When to Use These Prompts

Use productivity prompts when you need help with:
- Planning projects and tasks
- Creating meeting agendas
- Organizing workflows
- Setting goals and priorities
- Managing time effectively
- Making decisions
- Coordinating with teams

## Available Prompts

- **[Task Planning and Organization](task-planning.md)** - Break down complex tasks into manageable steps
- **[Meeting Agenda Creator](meeting-agenda.md)** - Create effective meeting agendas with clear outcomes

## Tips for Productivity Prompts

- Be realistic about available time and resources
- Specify deadlines and constraints
- Mention team size and roles if applicable
- Include context about current status
- Define clear goals and success criteria
- Consider dependencies and priorities
- Ask for specific formats (checklists, timelines, etc.)

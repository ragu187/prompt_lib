# Meeting Agenda Creator

## Category
Productivity

## Description
Create effective meeting agendas that ensure productive discussions and clear outcomes.

## Prompt

```
Create a meeting agenda for:

**Meeting Purpose:**
[PURPOSE]

**Meeting Type:**
[TYPE] (e.g., team sync, planning, review, brainstorm, decision-making)

**Attendees:**
[ATTENDEES_AND_ROLES]

**Duration:**
[TIME_ALLOCATED]

**Key Topics to Cover:**
1. [TOPIC_1]
2. [TOPIC_2]
3. [TOPIC_3]

**Decisions Needed:**
[DECISIONS_OR_OUTCOMES]

**Background/Context:**
[RELEVANT_CONTEXT]

Please create an agenda with:
1. Clear objectives
2. Time allocation for each item
3. Discussion points for each topic
4. Designated facilitators/presenters
5. Expected outcomes/deliverables
6. Action item capture process
```

## Variables

- `[PURPOSE]`: Why the meeting is being held
- `[TYPE]`: Category of meeting
- `[ATTENDEES_AND_ROLES]`: Who will attend and their roles
- `[TIME_ALLOCATED]`: Total meeting duration
- `[TOPIC_1/2/3]`: Main topics to discuss
- `[DECISIONS_OR_OUTCOMES]`: What needs to be decided or achieved
- `[RELEVANT_CONTEXT]`: Background information attendees need

## Example Usage

### Input
```
Create a meeting agenda for:

**Meeting Purpose:**
Plan Q1 product roadmap

**Meeting Type:**
Planning session

**Attendees:**
- Product Manager (Sarah)
- Engineering Lead (Mike)
- Design Lead (Lisa)
- Marketing Director (James)

**Duration:**
90 minutes

**Key Topics to Cover:**
1. Review customer feedback from Q4
2. Prioritize feature requests
3. Align on Q1 goals

**Decisions Needed:**
- Top 3 features for Q1
- Resource allocation
- Launch timeline

**Background/Context:**
Coming off successful Q4 launch, have 50+ feature requests from customers, need to balance new features with technical debt

Please create an agenda with:
1. Clear objectives
2. Time allocation for each item
3. Discussion points for each topic
4. Designated facilitators/presenters
5. Expected outcomes/deliverables
6. Action item capture process
```

### Expected Output
A structured agenda with time blocks, discussion questions, decision points, and clear outcomes for each section.

## Tips

- Share the agenda before the meeting
- Allocate realistic time for each item
- Include buffer time for discussions
- Assign clear owners for each topic
- Specify what preparation attendees need
- Include time for questions and action items
- Consider energy levels (don't put important items at the end)
